## Run Locally

Clone the project

```bash
  git clone https://github.com/mdakram09/Coding-Ninjas-Events-Page-Clone.git
```

Go to the project directory

```bash
  cd Coding-Ninjas-Events-Page-Clone
```

Start the server

```bash
  open index.html file in web browser
```
